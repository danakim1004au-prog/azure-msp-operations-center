# Project Roadmap

## Phase 1 - Portfolio Scaffold

- README, architecture, RBAC, support playbook.
- Terraform operations center resources.
- KQL query pack.
- Synthetic incident dataset.
- SLA breach model.
- RCA examples.

## Phase 2 - Real Azure Deployment

- Deploy Terraform to a test subscription.
- Connect diagnostic settings from sample App Service, SQL, and VM resources.
- Import Workbook JSON.
- Create alert rules and verify action group delivery.
- Attach Automation Account runbooks.

## Phase 3 - MSP/Multi-Tenant Expansion

- Add Azure Lighthouse delegation or simulated customer subscriptions.
- Split customers by resource group and tags.
- Add Sentinel analytics rules.
- Add Service Health and Resource Health monitoring.
- Add customer monthly operations review report.

## Phase 4 - Advanced Data Team Scope

- Export Log Analytics data to ADLS Gen2.
- Replace synthetic data with real incident history.
- Train a stronger model with time-window features.
- Add model drift monitoring.
- Store risk scores in Azure SQL or Storage Table.
